# Pandemic Analytics Dashboard 2.0

A modern, fully offline analytics dashboard that visualizes synthetic pandemic data through an interactive world map and data charts.

This project was developed as a demonstration of building research-oriented data infrastructure and visualization tools in the healthcare domain. It focuses on reliable offline data handling, clear visual communication, and an intuitive user interface.

---

## About the Project

The dashboard generates and analyzes synthetic pandemic data for multiple countries.  
It is built to simulate the kind of analytical environment used in health crisis management and preparedness projects.

The main objectives are-
- To showcase how data pipelines and visual dashboards can work completely offline.
- To illustrate cross country data comparison and temporal trend analysis.
- To provide a realistic example of data infrastructure relevant to research and hospital networks.

---

## Key Features

- Interactive world map to select and explore countries.
- Dynamic charts for new cases, deaths, vaccinations, and ICU patient trends.
- Country comparison view for side-by-side analysis.
- Synthetic data generation that avoids real patient or public data.
- Clean, professional interface that works in any environment.

---

## Technologies Used

**Framework:** Streamlit  
**Visualization:** Plotly  
**Data Processing:** Pandas, NumPy  
**Country Mapping:** PyCountry  
**Language:** Python 3.10+
